Probabilistic Programming Languages

Bayesian modeling with ppls

Probabilistic programming languages (PPL) are domain specific languages (DSL) with the objective of facilitating Bayesian analysis.

The Bayesian model is represented as a conditional simulation in the form of a probabilistic program.

A compiler then operates on the probabilistic program:

  • static analysis (by examining the program structure without running it)
  • statistical inference (by running the program within an inference procedure) - model comparison, sampling, learning parameters

Given an abstract problem, e.g. parameter estimation of a Bayesian model.

Example: Polynomial regression

model function poly(data: Real[][]): Int {
    assume n ~ Poisson(1.0); // sample the degree of the polynomial

    // Sample the coefficients from n + 1 (we need one coeff for const poly)
    let coeffs = repApply(addi(n, 1), function() {
        assume g ~ Gaussian(0.0, 1.0);
        return g;
    });

    // likelihood
    let sigma = 1.0; // noise term
    sapply(data, function(datum: Real[]) {
        let predictedY = polynomialFunction(coeffs, datum[1]);
        observe datum[2] ~ Gaussian(predictedY, sigma);
    });
    
    return n;
}

SMC Algorithm

SMC = Importance Sampling + Resampling

  1. Sample particles \(x_t\sim p(x_t|x_{t-1})\)
  2. Propagate particles until checkpoint \(y_t\)
  3. Resample particles based on particle likelihood \(p(y_t|x_t)\) and advance \(t\)
